\begin{DoxyItemize}
\item {\bfseries{Config\+Manager}}\+: Centralized configuration management
\item {\bfseries{Logger\+Manager}}\+: Structured logging with multiple output formats
\item {\bfseries{DB2\+Manager}}\+: IBM DB2 database operations via CLI
\item {\bfseries{MQManager}}\+: IBM MQ messaging operations via CLI
\item {\bfseries{CLI}}\+: Command-\/line interface for all operations
\item {\bfseries{Component\+Base}}\+: Base class for building components
\item {\bfseries{Component\+Runner}}\+: Component lifecycle management
\item {\bfseries{Component\+Registry}}\+: Component registration and discovery

\item {\bfseries{Standard Library Only}}\+: Uses only Python standard library modules
